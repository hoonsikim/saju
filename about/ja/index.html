四柱推命とは?

四柱推命(さしゅう、しちゅう)は文字通り「四つの柱」を意味します。出生年・月・日・時を天干と地支の組み合わせで読み解く韓国の伝統です。中国の八字(BaZi)とルーツを共有しますが、韓国で数百年にわたって精錬された独自の解釈体系を持ちます。

このサイトは本物を提供します — 実際の四柱、日主、五行のバランス、十神、蔵干、十二運星、空亡、十年大運まで。西洋占術を東洋風に装ったものではありません。

何が得られるか

  • 無料ベーシックリーディング — 四柱 + 日主 + 五行バランス + 短いAI解釈。即時。登録不要。命式計算はブラウザで行い、AI解釈はCloudflare WorkersとAnthropicを使用します。
  • $7 心層リーディング — 上記すべて + 蔵干・十二運星・空亡・十年大運を一つの物語に織り込んだAI長文解釈。メールで配信。

どのように動くか

決定論的な命式計算はブラウザで実行できますが、現在の無料リーディングとすべての有料・受注制作リーディングを含む全AI解釈では、生年月日・現地出生時刻・言語・選択した焦点・該当する伝統的な性別設定と派生命式項目をCloudflare WorkersとAnthropic Claude APIへ送信します。都市/国は無料またはEtsy経路で入力された場合に使われますが、Gumroad直接決済からは意図的に除外します。無料結果は再表示できるよう現在のページURLに生年月日・時刻・焦点を保持し、リセットで削除します。直接決済では現在の配信に必要な最小項目である生年月日・時刻・性別・言語・焦点をGumroad URLへ渡し、Gumroadが購入者メールを提供します。Sajuは有料決済用の出生情報コピーをブラウザストレージへ別途保存しません。直接配信の注文・リーディングデータはGumroad、Cloudflare Workers、Anthropic、Resendを通り、Etsyは独自の注文・メッセージ記録を保持します。出生データを販売したり、意図的に分析・広告オーディエンス・メーリングリスト・再利用学習データへ入れたりしません。Etsy注文の30日削除ツールが対象とするのは、文書化された保留を除くローカルの原出生データ作業ディレクトリのみで、プロバイダー・取引・メール・メッセージ記録は削除しません。Anthropicは標準API入出力を明示された例外を除き30日以内に削除し、既定ではモデル学習に使わないと説明しています。他のプロバイダーは各社の規約を適用します。

直接決済はGumroadで処理され、現在の自動メール経路はGumroad → Cloudflare Workers → Anthropic → Resendです。?paid=deepで戻ったこと自体は支払い完了の証明ではなく、決済画面とメールで配信状況を確認します。Etsyの受注制作購入は別の約束に従い、支払済み注文時刻から24時間以内に完成PDFをEtsy Ordersへアップロードします。

方法論(Methodology)

チャート計算は決定論的で再現可能です。オープンソースのlunar-javascriptライブラリ(v1.7.7)をベースに、韓国正統四柱推命の慣例に従います:

  • 年柱: 立春基準(旧正月ではない)
  • 月柱: 24節気の節入り基準
  • 日柱: 60甲子サイクル、子時(23:00–01:00)は翌日の日柱として処理(夜子時 = 翌日学派)
  • 時柱: 24時間 → 12地支マッピング
  • 蔵干、十二運星、空亡、大運、神煞(天乙貴人・桃花・駅馬): すべてlunar-javascriptのEightCharインターフェースで計算

真太陽時(LMT)経度補正は現在適用されていません — 出生地入力はリーディング文脈用です。分単位の精度が必要な場合(時柱境界ケース)は、有資格の鑑定士に相談してください。

AIモデルカード

現在の無料リーディングと有料・受注制作リーディングを含む全AI解釈は、以下を使用して生成されます:

  • モデル: Anthropic Claude Opus 4.8 (claude-opus-4-8)
  • 設定上の最大出力トークン: 8,192
  • 再現性: 同じ生年月日 + 同じreadingType → 類似の解釈 (LLMは確率的、同一出力は保証されません)

  • モデルへの入力: 生年月日・現地時刻・該当する伝統的な性別設定・言語・readingType (general/career/love/wealth)と、Cloudflare Workerが計算した決定論的チャート構造。都市/国は無料またはEtsy経路で入力された場合に含まれますが、現在のGumroad直接決済からは除外されます。該当する原出生項目に加え、四柱・日主・五行・十神・蔵干・十二運星・空亡・大運・歳運の文脈が含まれます。

チャート自体(四柱・五行・十神・蔵干・十二運星・空亡・大運・神煞・歳運)は決定論的でGitHubソースから検証可能です。LLMはその上に自然言語の解釈レイヤーを加えるだけです。
